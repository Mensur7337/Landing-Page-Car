

# 🏎️ McLaren - Modern Landing Page Experience

McLaren is a modern landing page project for a 2026 luxury supercar collection.  
This project focuses on user experience, speed, and clean design.

---

## 🎯 Project Overview

- **Brand:** McLaren Automotive  
- **Theme:** Dark & Orange (Papaya Orange + Carbon Black)  
- **Goal:** Create a fast and smooth video-based experience  

---

## ✨ Features

- 🖱️ **Smooth Scroll Navigation**  
  - Full screen sections (100vh)  
  - Auto active navigation link  

- 🎬 **Dynamic About Section**  
  - Click “Explore” to see model details  
  - Content updates without page reload  
  - Smooth scroll to detail section  

- 🎥 **Video System**  
  - Hover to play preview videos  
  - Custom video player (play / pause)  
  - Trailer switching system  

---

## 🛠️ Technologies

- HTML5 (Semantic structure)  
- CSS3 (Scroll-snap, Flexbox, Grid)  
- Vanilla JavaScript  
- Intersection Observer API  

---

## ⚡ Performance & Accessibility

- High performance (fast loading videos)  
- Responsive design  
- Accessible structure (alt tags, contrast colors)  
- SEO friendly HTML
